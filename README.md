# EchoMind AI

EchoMind AI is a neural-inspired second brain that captures your spoken thoughts in real time, transcribes them, extracts structured memories, and stores everything in a futuristic searchable vault.

## What it does
- Records audio from mobile or web
- Streams audio to the backend in real time
- Transcribes speech into text
- Extracts Tasks, Facts, and Creative Ideas
- Displays memories in a searchable dashboard

## MVP Scope
- Expo capture app
- Next.js dashboard
- WebSocket audio stream
- Whisper transcription
- Gemini memory extraction
- Database-backed vault

## Getting Started
1. Review the PRD and system design.
2. Build the backend API and audio pipeline.
3. Build the capture UI and vault UI.
4. Connect transcription and extraction services.
5. Deploy to staging and production.

## Notes
This repository is designed for an AI memory assistant prototype and can be expanded into a full knowledge management platform.
