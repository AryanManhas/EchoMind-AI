# SKILL_DESIGN — EchoMind AI

## Objective
Design a futuristic, calm, and intelligent interface that makes the product feel like a living second brain.

## Design Goals
- Make capture feel magical
- Make search feel instant and organized
- Make memories feel valuable and permanent
- Create a strong visual identity around the Neural Orb

## Visual Language
### Mood
- Futuristic
- Liquid
- Calm
- Intelligent
- Premium

### Color Direction
- Deep dark backgrounds
- Subtle gradients
- Cool accent tones
- Soft glow highlights

### Typography
- Large expressive headings
- Clean readable body type
- Strong contrast between labels and content

## Key Visual Motifs
- Orbital motion
- Neural pulses
- Data flow lines
- Glass cards
- Soft blurred light

## Layout Principles
- Use structured grids
- Keep spacing generous
- Separate capture and memory zones clearly
- Highlight the main action prominently

## Motion Principles
- Gentle and responsive animations
- No distracting looping motion
- State changes should have clear transitions
- Audio-reactive movement should mirror energy, not chaos

## Component Style Guidance
### Buttons
- Rounded
- Clear active/hover states
- Primary action should be obvious

### Cards
- Soft shadows
- Layered depth
- Clear hierarchy of title, metadata, and content

### Search and Filters
- Prominent placement
- Easy to scan
- Clear selected states

## Design Deliverables
- Capture screen mock
- Vault dashboard mock
- Memory detail mock
- Component style guide
- Color and typography palette
- Motion and interaction references

## Design Review Checklist
- Is capture understandable in one glance?
- Is the vault readable at high density?
- Does the Orb feel alive without overwhelming the interface?
- Does the app feel premium on both mobile and web?
